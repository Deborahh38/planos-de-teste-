# 🧪 Cenários - Cadastro

## Cenário 1: Cadastro com sucesso
Dado que o usuário está na tela de cadastro  
Quando preencher os dados corretamente  
Então o cadastro deve ser realizado  

## Cenário 2: Email já existente
Dado que o usuário está na tela de cadastro  
Quando informar um email já cadastrado  
Então deve exibir erro  

## Cenário 3: Senha fraca
Dado que o usuário está na tela de cadastro  
Quando inserir senha fraca  
Então deve exibir alerta
