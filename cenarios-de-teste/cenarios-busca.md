# 🧪 Cenários - Busca

## Cenário 1: Busca com resultado
Dado que o usuário está na tela de busca  
Quando pesquisar por um termo válido  
Então deve retornar resultados  

## Cenário 2: Busca sem resultado
Quando pesquisar termo inexistente  
Então deve exibir mensagem  

## Cenário 3: Busca vazia
Quando não inserir texto  
Então deve impedir a busca
