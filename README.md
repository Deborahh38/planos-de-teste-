# planos-de-teste-
# 📌 Portfólio de QA - Planos de Teste

Este repositório contém exemplos práticos de documentação de testes, com foco em testes funcionais, exploratórios e validação de regras de negócio.

## 🧪 Conteúdo
- Planos de teste
- Cenários de teste (BDD)
- Casos de teste
- Massa de dados
- Evidências (simuladas)

## 🚀 Objetivo
Demonstrar habilidades em:
- Escrita de cenários de teste
- Análise de requisitos
- Identificação de cenários críticos
- Organização de documentação de QA

## 🛠️ Ferramentas e técnicas
- BDD (Gherkin)
- Testes manuais
- Postman
- Testes exploratórios

## 📂 Estrutura
- `planos-de-teste/` → Estratégia de testes
- `cenarios-de-teste/` → Cenários em BDD
- `casos-de-teste/` → Casos detalhados
- `massa-de-teste/` → Dados para execução
