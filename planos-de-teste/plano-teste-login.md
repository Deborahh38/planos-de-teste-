# 📋 Plano de Teste - Login

## 📌 Aplicação
Sistema: App Mobile  
Funcionalidade: Login  

## 🎯 Objetivo
Validar o acesso do usuário ao sistema com diferentes tipos de credenciais.

## 🔍 Escopo
- Login com email e senha
- Login inválido
- Validação de campos

## 🧪 Tipos de Teste
- Teste funcional
- Teste exploratório

## ✅ Critérios de Entrada
- Usuário previamente cadastrado
- Sistema disponível

## ❌ Critérios de Saída
- Todos os cenários executados
- Bugs críticos resolvidos

## ⚠️ Riscos
- Instabilidade do sistema
- Falha de integração com API

## 📊 Massa de Dados
- Email válido
- Email inválido
- Senha correta/incorreta
